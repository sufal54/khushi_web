Put your release files in this folder.

Examples:
- public/downloads/khushi-linux.AppImage  →  /downloads/khushi-linux.AppImage
- public/downloads/khushi-linux.deb       →  /downloads/khushi-linux.deb
- public/downloads/khushi-linux.rpm       →  /downloads/khushi-linux.rpm

Then update the download buttons in:
- src/components/LandingPage.tsx
- src/app/download/linux/page.tsx
